# 《代码修仙录》设定总览

## 📖 基本信息

- **书名：** 代码修仙录
- **类型：** 玄幻（穿越+修仙+后宫+热血成长）
- **卷数：** 5卷
- **章节：** 每卷30章，共150章
- **目标字数：** 100万字

---

## 👥 人物设定

### 男主角：林辰

- **身份：** 地球996程序员 → 青云宗外门弟子 → 修仙界霸主
- **核心：** 程序员思维，逻辑清晰，善于发现和利用漏洞
- **系统：** 上古大能遗落的半吊子代码修仙系统，靠写代码补全
- **成长线：** 被压迫 → 系统觉醒 → 反击 → 崛起 → 称霸

### 女主角（后宫）

1. **苏清雪** - 青云宗大师姐，高冷冰山，外冷内热
2. **月灵溪** - 青云宗小师妹，治愈系，林辰的白月光
3. **萧媚儿** - 青云宗魅惑魔女，主动倒追
4. **叶紫萱** - 紫霞宗天骄，相爱相杀
5. **楚凝霜** - 修仙界第一美女，神秘，最终道侣

### 反派

- **赵天宇** - 青云宗天才，前期PUA压迫林辰，后期被打脸

---

## 🌍 世界观设定

### 境界体系

练气期 → 筑基期 → 金丹期 → 元婴期 → 化神期 → 渡劫期 → 飞升期

### 主要宗门

- **青云宗** - 主角加入的宗门
- **紫霞宗** - 敌对宗门

### 代码修仙系统特色

- **Bug发现：** 能发现修仙界的"漏洞"
- **Debug：** 能修复/破坏别人功法
- **代码复刻：** 复制功法并用代码优化
- **系统升级：** 靠写代码让系统进化

---

## 📈 故事主线

1. **穿越：** 地球996程序员林晨加班猝死，穿越到修仙界
2. **被压迫：** 成为青云宗外门弟子，被赵天宇PUA和压迫
3. **系统觉醒：** 绝境中觉醒半吊子代码修仙系统
4. **反击：** 利用系统开挂，开始反击赵天宇
5. **崛起：** 泡上各大美女天骄，在宗门内崛起
6. **称霸：** 逐步成长，最终成为守护修仙界的霸主

---

## 📂 详细文件位置

所有详细设定都在 `SPECS/` 目录下：

- `SPECS/characters/` - 7个人物的详细设定
- `SPECS/world/` - 完整世界观设定
- `SPECS/meta/` - 故事概念文件
